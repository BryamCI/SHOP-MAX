# SHOP-MAX

Herramientas utilizadas en este proyecto son:
- Netbeans IDE 25.
- Tomcat 10.
- Conector de base datos: mysql-connector-j-9.6.0.jar
- Java versión 21.

Estructura del proyecto:
a) css: carpeta de los estilos
b) images: carpeta de imagenes
c) js: carpeta de código JavaScript
d) dashboard.jsp: sistema
e) index.html: página principal
f) En Source Packages tenemos:
  - Controller
  - Entity
  - Model
  - Utils
g) database: shopmax

Aún falta estructurar la arquitectura, ya que se trabajará con la arquitectura DAO.

Base de datos:
- La base de datos se llama "shopmax", la encuentras en la carpeta "database", la descargas y la importa en Mysql.
