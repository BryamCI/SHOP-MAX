# SHOP-MAX

Herramientas utilizadas en este proyecto son:
- Netbeans IDE 25.
- Tomcat 10.
- Conector de base datos: mysql-connector-j-9.6.0.jar
- Java versión 21.

Estructura del proyecto:
-css: carpeta de los estilos
- images: carpeta de imagenes
- js: carpeta de código JavaScript
- dashboard.jsp: sistema
- index.html: página principal
- En Source Packages tenemos: Controller, Entity, Model, Utils
- database: shopmax

Aún falta estructurar la arquitectura, ya que se trabajará con la arquitectura DAO.

Base de datos:
- La base de datos se llama "shopmax", la encuentras en la carpeta "database", la descargas y la importa en Mysql.
